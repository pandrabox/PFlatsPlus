# アンインストール方法
- アバターからFlatsPlusプレハブを削除すれば、ほぼすべての影響は消えますが、より確実に削除する必要がある場合は次の通り実施してください。
- Assets/Pan/FlatsPlusフォルダを削除してください。
  - カスタム表情を作成した場合は、ここに保存されていますのでご注意ください。
- VCCでManage Projectを開いてください。
  - FlatsPlus、PandraVaseを削除してください。
